Goal: I'm building an extension that serves users discounted giftcards from providers such as cardcash, carddepot, etc. To serve this our app has a database of the url's of these discounted gifts (respective to each provider) and when the time comes to fetch them we make sure we send back the discounted gift card with the maximum discount. To supplement this and bring in more users we're going to build a directory where we have every brand we support and every provider support for each brand and their discount. Then the user can click on which provider they want and they will be redirect to their site where they can checkout.
